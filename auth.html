📧 Email & SMS setup guide
One-time configuration to enable real OTP delivery
📧 Email OTP via EmailJS
1. Go to emailjs.com → free signup
2. Add an Email Service (Gmail works)
3. Create a Template — add variables: {{otp}}, {{to_email}}, {{to_name}}
4. Copy your Service ID, Template ID, Public Key
5. Paste them in auth.html where marked ← YOUR KEY
📱 SMS OTP via Fast2SMS
1. Go to fast2sms.com → free signup (Indian numbers)
2. Go to Dev API → copy your API Key
3. Paste it in auth.html where marked ← YOUR KEY
4. Free plan gives ₹50 credit (~100 SMS)
